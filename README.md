# MEAN ToDo (ES2015)
Full-stack todo app for the lolz.

## Getting Started

#### Prerequisites

- npm
- node
- nodemon
- mongodb

### Start Mongod

```
$ mongod
```

### Start the Server

```
/back-end $ npm install
/back-end $ npm start
```

### Run the App

```
/front-end $ npm install
/front-end $ npm start
```
