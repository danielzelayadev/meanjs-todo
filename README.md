# MEAN ToDo
Full-stack todo app for the lolz.

## Getting Started

#### Prerequisites

- npm
- node
- nodemon
- mongodb
- browser-sync or anything to run a local server for your client files

### Start Mongod

```
$ mongod
```

### Start the Server

```
/back-end $ npm install
/back-end $ npm start
```

### Running the App

```
/front-end $ browser-sync start --server
```
